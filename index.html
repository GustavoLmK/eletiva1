Aprendendo a Programar

HTML

HTML é a linguagem que cria o conteúdo de um site. Usa tags para organizar o texto e as imagens.

CSS

CSS deixa o site bonito e colorido. Muda as cores, tamanhos e posições das coisas.

JavaScript

JavaScript faz o site fazer coisas. Quando você clica em um botão, JavaScript faz algo acontecer.
